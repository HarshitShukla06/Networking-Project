🎓 College Network Design using Cisco Packet Tracer
📌 Project Overview

This project focuses on the design and implementation of a college campus network using Cisco Packet Tracer. It simulates a real-world network infrastructure including multiple departments, routing, switching, and secure communication between devices.

The aim is to demonstrate how a scalable, efficient, and secure network can be built for an educational institution.

🛠️ Tools & Technologies
Cisco Packet Tracer
Networking Concepts:
VLAN (Virtual LAN)
Routing Protocols (Static/Dynamic)
Switching
IP Addressing & Subnetting
Network Security Basics
🏗️ Network Design

The network is divided into multiple departments such as:

Administration
Computer Lab
Library
Faculty Rooms

Each department is connected through switches and routers, ensuring proper communication and network segmentation.

⚙️ Features
✅ VLAN-based network segmentation
✅ Inter-VLAN routing
✅ Efficient IP addressing scheme
✅ Secure communication between departments
✅ Scalable network design
✅ Real-world simulation using Cisco Packet Tracer
🔌 Network Components Used
Routers
Switches
PCs (End Devices)
Servers (Optional)
📊 Working Principle
Devices in each department are grouped using VLANs
Switches forward data within VLANs
Routers enable communication between VLANs (Inter-VLAN Routing)
Proper IP addressing ensures smooth communication
The network is tested using ping commands to verify connectivity
